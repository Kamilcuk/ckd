# ckd library

Implementation of `Towards Integer Safety` http://www.open-std.org/jtc1/sc22/wg14/www/docs/n2792.pdf on top of GNU macros.

# Building

Prerequisites:
- cmake
- python3
- python3-jinja2

Build with cmake.
